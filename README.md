# Project 3: Spots

## Overview

Spots is a responsive image-sharing project that demonstrates DOM manipulation, form validation, modal interactions, and accessible UI controls built with vanilla JavaScript and modular CSS (Flat BEM). Users can edit their profile, create new posts (cards), like and delete cards, and preview images in a modal.

## Features

- Responsive layout with BEM-structured CSS files
- Generate initial cards from a template and JavaScript array
- Create new cards with client-side validation
- Edit profile with pre-filled form values
- Image preview modal with caption and accessible close controls
- Like and delete actions on cards
- Modals close with overlay click and Escape key
- Form validation implemented in `scripts/validation.js` with a configuration object

## Tech

- HTML5
- CSS3 (BEM, modular blocks)
- JavaScript (ES modules)
